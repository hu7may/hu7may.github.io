Vue.js入门

Posted on 2019-02-27

最近开始着手开发老师交代的课程系统，虽然活都是一个人干，还是准备前后端分离，毕竟是主流趋势。所以要学习一点Vue。

环境搭建

npm install -g vue-cli

初始化操作：

vue init webpack myProject

运行：

cd myProject

npm install

npm run dev

开发完成后:npm run build会编译源码会生成发布代码到dist目录
