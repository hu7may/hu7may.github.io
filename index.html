正则表达式

Posted on 2018-10-19 | In programing notes

特殊符号与字符：

r1|r2 匹配r1或者r2 foo|bar

. 匹配任何字符（\n除外） b.b

^ 匹配字符串起始部分 ^Dear

$ 匹配字符串终止部分 /bin/*sh$ ^subject:hi$ 任何有单独的字符串subject:hi构成的字符串。

匹配0次或多次前面出现的正则表达式 [A-Za-z0-9]

+一次或多次 [a-z]+\ .com

? 0次或一次

{N}匹配n次前面出现的表达式 [0-9]{3}

{M,N}匹配m-n次前面出现的表达式

[…]匹配来自字符集的任意单一字符

[.x-y..] 【】 [0-9],[A-Za-z]

\d任何十进制字符，与[0-9]一致，（\D不匹配任何数字）data\d+.txt

\w任何字母以及数字，与[A-Zz-z0-9]相同。（\W与之相反）

git 常用命令

Posted on 2018-06-19 | In programing notes

git常用命令

初始化仓库：git init

添加：git add readme.txt

提交：git commit -m "description"

创建同名远程仓库

关联远程库：git remote add origin git@github.com:michaelliao/learngit.git

或者从远程库克隆 ： git clone https://......

将本地库的全部内容推到远程库：git push -u origin master

从现在起，只要本地作了提交，就可以通过命令：git push origin master
